GET /items

Retrieve all items

JSON response body fields

Name Type Description
id String The unique id of the item retrieved
brand String The brand name of the item retrieved
model String The model name of the item retrieved
os String The operating system of the item retrieved
image String The image URL of the item retrieved
screensize Number The screen size of the item retrieved

POST /items

Add an item

All request body fields are required

JSON request body fields

Name Type Description
id String The unique id of the item to be added
brand String The brand name of the item to be added
model String The model name of the item to be added
os String The operating system of the item to be added
image String The image URL of the item to be added
screensize Number The screen size of the item to be added

GET /items/:id

Retrieve an item by its id

Path parameters

Name Type Description
id String The unique id of the item to be retrieved

JSON response body fields

Name Type Description
id String The unique id of the item retrieved
brand String The brand name of the item retrieved
model String The model name of the item retrieved
os String The operating system of the item retrieved
image String The image URL of the item retrieved
screensize Number The screen size of the item retrieved

PUT /items/:id

Update the data of an item by its id

Path parameters

Name Type Description
id String The unique id of the item to be updated

JSON request body fields

All request body fields are optional

Name Type Description
brand String The brand name of the item to be updated
model String The model name of the item to be updated
os String The operating system of the item to be updated
image String The image URL of the item to be updated
screensize Number The screen size of the item to be updated

DELETE /items/:id

Delete an item by its id

Path parameters

Name Type Description
id String The unique id of the item to be deleted
