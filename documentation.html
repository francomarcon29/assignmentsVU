GET /items POST /items GET /items/:id PUT /items/:id DELETE /items/:id

GET /items

Retrieve all items

JSON response body fields

Name Type Description
id Integer The unique id of the item retrieved
brand String The brand name of the item retrieved
model String The model name of the item retrieved
os String The operating system of the item retrieved
image String The image URL of the item retrieved
screensize Number The screen size of the item retrieved

Response codes

Response code Meaning
200 Succesfully retrieved all items
500 Server encountered an error while trying to retrieve all items

POST /items

Add an item

All request body fields are required

JSON request body fields

Name Type Description
brand String The brand name of the item to be added
model String The model name of the item to be added
os String The operating system of the item to be added
image String The image URL of the item to be added
screensize Number The screen size of the item to be added

Response codes

Response code Meaning
200 Succesfully added an item to the database
400 Client sent invalid JSON in the request body
500 Server encountered an error while trying to add an item to the database

GET /items/:id

Retrieve an item by its id

Path parameters

Name Type Description
id Integer The unique id of the item to be retrieved

JSON response body fields

Name Type Description
id Integer The unique id of the item retrieved
brand String The brand name of the item retrieved
model String The model name of the item retrieved
os String The operating system of the item retrieved
image String The image URL of the item retrieved
screensize Number The screen size of the item retrieved

Response codes

Response code Meaning
200 Succesfully queried database for an item with the given id, response body can be an empty list if there are no such items
500 Server encountered an error while trying to get the item with the given id

PUT /items/:id

Update the data of an item by its id

Path parameters

Name Type Description
id Integer The unique id of the item to be updated

JSON request body fields

All request body fields are optional

Name Type Description
brand String The brand name of the item to be updated
model String The model name of the item to be updated
os String The operating system of the item to be updated
image String The image URL of the item to be updated
screensize Number The screen size of the item to be updated

Response codes

Response code Meaning
200 Succesfully modified the item with the given id
400 Client sent invalid JSON in the request body
500 Server encountered an error while trying to modify the item with the given id

DELETE /items/:id

Delete an item by its id

Path parameters

Name Type Description
id Integer The unique id of the item to be deleted

Response codes

Response code Meaning
200 Succesfully deleted the item with the given id
500 Server encountered an error while trying to delete the item with the given id
